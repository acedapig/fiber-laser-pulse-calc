Changelog

Full version history — what changed and when.

v5.11
  • Fixed Parameter Converter swap button — manufacturer swapped correctly but model went blank; root cause was model values being read after dropdown rebuilds had already cleared them
v5.10
  • Clarified below-cutoff MOPA physics — E/pulse and peak power are constant below cutoff while avg power scales linearly with frequency; this is correct MOPA behaviour, not a bug
  • Improved freq note and Avg Power, Pulse Energy, and Peak Power tooltips to explain this explicitly
  • Removed duplicate entries for YDFLP-E-80-M7-M-R and YDFLP-E-100-M7-M-R which had been added twice under different IDs with identical pulse tables
v5.9
  • Added ★ Favourites system — star any laser in the calculator to pin it to a Favourites group at the top of the brand dropdown; persists across sessions
  • Added last used persistence — calculator remembers your selected laser and restores it on next load
  • Fixed unfavourite bug — removing last favourite now correctly drops back to the laser's real brand; removing from Favourites view with others remaining stays in Favourites
v5.8
  • Major laser database expansion — 53 total models across JPT, Raycus, and Max Photonics
  • JPT: added E-LP, M6+, M7, M7-S, M8, C-M7, and L1 series across 20–200W (17 new MOPA models)
  • Raycus: added full MX MOPA series (P20MX–P200MX, P200S) and expanded Q-switched lineup (P20QB, P20QS, P30QB, P30QS, P50QB, P60QB, P70Q, P70QA, P100Q, P100QA)
  • Max Photonics: added MFP-10W, MFP-30W, MFP-50W, MFP-100W, MFP-200W; renamed MFP-50W/X → MFP-50X
  • Corrected tauMin for RFL-P30Q (90→120 ns) and RFL-P50QB (90→120 ns) based on spec sheet verification
  • All models sorted by wattage ascending within each brand
v5.7
  • Added 📋 Changelog as an in-app modal — version history is now embedded in the HTML file itself, accessible from the footer next to the version badge
  • Removed Changelog as a separate tab — stays on whichever tab you're already using
v5.6
  • Added Effective Avg Power at Material row to both Source and Destination panels in the Converter — shows actual watts at the material (pulse energy × frequency) so users can understand why power% values differ between lasers while delivering equivalent energy
v5.5
  • Added ⚡ Send to Calculator → button at the bottom of Solved Settings — transfers all destination parameters to the Calculator tab and switches to it, so users can fine-tune from the solved result
  • Header briefly flashes cyan on transfer as visual confirmation
v5.4
  • Solver now penalizes pulse width deviation for MOPA→MOPA conversions — strongly prefers matching source pulse width rather than finding mathematically equivalent solutions at different pulse widths
  • Pulse width penalty weight is source-type aware: applied for MOPA→MOPA, zero for Q-SW→MOPA
v5.3
  • Added Match Confidence score (0–100%) to Match Results — color-coded progress bar with plain English label
  • Green ≥80%, Amber ≥50%, Red <50%. Uses exponential decay: confidence = 100 × e^(−score × 2)
  • Shown on all three alternative solution tabs
v5.2
  • Added frequency proximity penalty to solver — prevents returning technically correct but practically useless solutions at wildly different frequencies
  • Added low scan speed penalty (below 10 mm/s is flagged as suspicious)
  • Fixed solver returning 1 kHz / 20% on 60W when correct answer was 80 kHz / 33.3%
v5.1
  • Pulse width priority is now source-type aware — MOPA→MOPA starts at source pulse width and radiates outward; Q-SW→MOPA uses 200ns → 100ns priority
  • Three alternative solutions now use genuinely different scoring weights (Fluence Priority, Overlap Priority, Balanced)
  • Added power pre-check warning when destination rated power is below source effective power
v5.0 — Parameter Converter
  • Added ⇄ Parameter Converter tab — converts a known working parameter set from one laser source to an equivalent set for a different laser
  • Source side: full parameter entry (laser, optics, frequency, power%, pulse width, speed, LPI/LI). Calculated Targets update live
  • Destination side: laser + optics only. Solver finds frequency, power%, pulse width, speed, and line interval to match source targets
  • Destination auto-populates from active Calculator selection on tab switch
  • ⇄ Swap button flips all values between source and destination
  • Exact match shown as single result; no exact match auto-generates 3 alternatives
  • Match table with ✓/⚠/✗ badges for Fluence, Pulse Energy, Pulse Overlap, Line Overlap, Peak Power
  • Optional pulse width override for MOPA destination lasers
